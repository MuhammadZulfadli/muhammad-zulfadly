# Planning: Personal Website Landing Page (Frontend Engineer)

## Deskripsi Tugas

Membangun sebuah landing page (satu halaman utama) untuk website personal portofolio. Fokus utama website ini adalah untuk menampilkan profil, keahlian, dan rekam jejak saya sebagai seorang Frontend Engineer.

## Tech Stack Target

- **Framework/Library Utama:** Vue (atau Nuxt.js)
- **Bahasa:** JavaScript
- **Styling:** Tailwind CSS
- **Environment & Version Control:** Linux, Git

## Struktur Konten (High-Level)

Halaman utama harus dibagi ke dalam beberapa section berikut:

1. **Hero / Background Section**
   - Tampilkan perkenalan diri secara profesional.
   - Deskripsikan **Background saya sebagai Frontend Engineer** (siapa saya, fokus, dan apa yang bisa saya lakukan).

2. **Skills Section**
   - Tampilkan daftar teknologi atau keahlian utama.
   - Berikan sorotan khusus pada tech stack andalan: **Javascript, Vue, Nuxtjs, Tailwind CSS, Git, Linux**.

3. **Experience Section**
   - Menampilkan riwayat pekerjaan (timeline atau list).
   - Cukup cantumkan peran sebagai Frontend Engineer, tempat kerja, durasi, dan ringkasan tanggung jawab.

4. **Projects Section**
   - Etalase untuk menampilkan portofolio proyek.
   - Buat dalam bentuk visual yang menarik (misal: Card layout), mencakup nama proyek, deskripsi singkat, dan stack yang digunakan.

5. **Contact / Footer Section**
   - Sediakan informasi kontak agar pengunjung bisa terhubung (Email, LinkedIn, GitHub).
   - Pastikan diletakkan di bagian bawah dengan desain yang tidak berlebihan.

## Panduan Implementasi (Untuk Programmer/AI)

- **Modularitas:** Buat setiap section menjadi komponen Vue yang terpisah (misalnya `HeroSection.vue`, `ProjectCard.vue`) untuk menjaga kebersihan kode.
- **Responsif:** Gunakan sepenuhnya utilitas Tailwind CSS untuk memastikan tampilan sempurna di _mobile_ maupun _desktop_.
- **Data Statis:** Gunakan data _dummy_ atau _hardcoded_ terlebih dahulu untuk mengisi konten. Fokuskan pekerjaan pada struktur layout, pewarnaan, margin/padding, dan estetika secara keseluruhan.

Silakan kerjakan implementasi ini berdasarkan kerangka kerja Vue yang sudah tersedia.
